Introduction

This guide is meant to help speed development of sites using a core theme. Additional themes will be added in the future to create more flexibility in web development.
Each section provides samples of each customization. As the site develops I'll include interactive elements to change themes or various elements within them.

Color themes

Try to limit the core colors of a site to 5 or 6 colors.
You can use color pallet generators to help create new pallets and see how they look.
Each color should display the corresponding code below it so you can easily insert it into the site.
To make the site more accessible, it's a good idea to test your color combinations contrast to make it easier to read.
You should also see how your color pallette will look with various types of color blindness. I recommend using Coolors.co, which will allow you to generate a color pallette and then filter it for many types of color blindness.

Fonts

Fonts should include web-safe backups in case the desired font isn't installed on a user's device. This guide will display the primary font samples along with the web-safe backup font samples. This site will also include "accent" fonts, used for special elements such as buttons, table headings, etc.
